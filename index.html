이런 점이 다릅니다

  • 화면 어디든 떠 있는 작은 동그라미 타이머 — 유튜브, 게임, 어떤 앱 위에든
  • 전체 시간 + 부분 시간 동시 표시 — 큰 흐름과 짧은 단위를 함께 봄
  • 부분 알람은 설정한 주기마다 전체 시간 끝까지 자동 반복 (예: 30분/10분 → 10분, 20분에 부분 알람 + 30분에 최종)
  • 다이얼 위 작은 노란 점이 알람 위치를 미리 보여줌, 지나면 흐려짐
  • 부분 알람 (가벼운 차임) / 최종 알람 (묵직한 경보) 두 톤이 다름
  • 알람 울릴 때 동그라미 한 번 누르면 즉시 멈춤
  • 드래그로 위치 이동, 길게 누르면 설정 화면
  • 어두운 테마 / 깔끔한 카드형 설정

설치 방법 (안드로이드 8 이상)

1
위 "앱 다운로드" 버튼을 누르세요 앱 설치 파일이 다운로드 폴더로 저장됩니다.
2
알림 또는 "내 파일"에서 받은 앱 파일을 누르세요 처음이면 "이 출처에서 설치 허용" 안내가 뜹니다 — 설정에서 토글 ON 후 뒤로.
3
설치 누르기 "Play Protect 검사" 떠도 자세히 보기 → 무시하고 설치 OK.
4
앱 열고 "다른 앱 위에 표시" 권한 허용 권한 화면에서 "업글 타이머" ON → 뒤로.
5
전체 시간 / 부분 시간 입력 → "타이머 시작" 기본값 30분 / 10분. 빠른 선택 버튼도 있어요.

조작법

  • 동그라미 한 번 누르기 = 시작 / 정지
  • 동그라미 드래그 = 위치 옮기기
  • 동그라미 길게 누르기 = 설정 화면으로 돌아가기
  • 알람 울릴 때 한 번 누르기 = 알람 즉시 끔

이런 분들께

  • 유튜브 / 게임 시간을 스스로 조절하고 싶은 분
  • 아이의 시청 시간을 옆에서 같이 보고 싶은 부모
  • 포모도로 (25-5분) 같은 시간 인터벌로 작업하는 분
  • 운동 / 명상 / 단기 집중에 리듬을 잡고 싶은 분
