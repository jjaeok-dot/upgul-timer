## 긴 설명 (4000자 이내)

```
시간을 주도하는 사람을 위한 오버레이 타이머

업글 타이머는 화면 어떤 앱 위에든 떠 있는 작은 동그라미 타이머입니다. 유튜브를 보든, 게임을 하든, 책을 읽든 — 사용한 시간을 잊지 않게 도와줍니다.

■ 핵심 기능
• 화면 어디든 떠 있는 작은 동그라미 (드래그로 위치 자유)
• 전체 시간 + 부분 시간 동시 표시 (예: 30분 안에 10분마다 알람)
• 부분 알람은 설정한 주기마다 전체 시간 끝까지 자동 반복
• 다이얼 위 노란 점이 다음 알람 위치를 미리 보여줌
• 부분 알람(가벼운 차임) / 최종 알람(묵직한 경보) 두 톤이 분명히 다름
• 알람 울릴 때 동그라미 한 번 누르면 즉시 멈춤
• 어두운 테마와 깔끔한 카드형 설정 화면

■ 이런 분들께
• 유튜브 / 게임 시간을 스스로 조절하고 싶은 분
• 자녀의 시청 시간을 옆에서 같이 보고 싶은 부모
• 포모도로(25-5분) 같은 시간 인터벌로 작업하는 분
• 운동 / 명상 / 단기 집중에 리듬을 잡고 싶은 분

■ 조작법
• 동그라미 한 번 누르기 = 시작 / 정지
• 드래그 = 위치 옮기기
• 길게 누르기 = 설정 화면으로 돌아가기
• 알람 중 한 번 누르기 = 알람 즉시 끔

■ 데이터 / 권한
이 앱은 어떤 개인정보도 수집·저장·전송하지 않습니다. 광고 SDK, 분석 SDK 등 외부 라이브러리도 포함되지 않습니다. 인터넷도 사용하지 않습니다. 단순히 시간 관리를 돕는 도구입니다.

"다른 앱 위에 표시" 권한은 동그라미 타이머를 화면에 띄우기 위해, "포그라운드 서비스" 권한은 타이머가 백그라운드에서도 정확히 동작하도록 유지하기 위해 사용됩니다.

■ 주의
이 앱은 유튜브 등 다른 앱을 차단하지 않습니다. 사용 시간을 시각적으로 보여주는 보조 도구입니다. 차단이 필요하면 갤럭시의 자녀 보호 / 디지털 웰빙 / Family Link를 함께 사용하세요.

만든이: upgul
```

## 권한 정당성 (Permission Rationale)

스토어 폼에 권한별로 사용 이유 적는 칸이 나오면:

### `SYSTEM_ALERT_WINDOW` (다른 앱 위에 표시)
```
이 앱의 핵심 기능은 다른 앱 위에 작은 동그라미 타이머를 띄우는 것입니다. 사용자가 유튜브, 게임, 다른 작업 앱을 사용하는 동안 시간을 시각적으로 확인할 수 있도록 본 권한이 필수적입니다. 사용자가 앱 안에서 명시적으로 허용 버튼을 눌러야만 권한이 활성화됩니다.
```

### `FOREGROUND_SERVICE` / `FOREGROUND_SERVICE_SPECIAL_USE`
```
타이머가 사용자가 다른 앱을 사용하는 동안에도 정확한 시간을 유지하려면 포그라운드 서비스로 동작해야 합니다. 사용자에게는 항상 알림창에 "업글 타이머 실행 중" 알림이 표시되며, 사용자가 앱이나 알림에서 종료를 선택하면 즉시 종료됩니다. 백그라운드에서 임의로 데이터를 수집하거나 외부 통신을 수행하지 않습니다.
```

### `POST_NOTIFICATIONS`
```
포그라운드 서비스의 필수 알림(타이머가 실행 중임을 알리는 영구 알림)을 표시하기 위해 사용됩니다. 별도의 푸시 알림이나 마케팅 알림은 보내지 않습니다.
```

## 첫 등록 시 주의사항 / 자주 막히는 부분

1. **본인 인증**: 휴대폰 본인인증 또는 PASS 앱 필요. 외국인등록번호 가능.
2. **연령 등급**: "자가진단" 선택 → 폭력/성/도박 등 모두 "없음" → 만 4세 이상으로 자동 책정됨.
3. **국가/언어**: 한국어를 기본으로 등록하고, 영어는 선택. 영어 미등록도 OK.
4. **콘텐츠 검수 기준**: 갤럭시 스토어는 다음을 자주 봄
   - APK가 정상 서명되었는지 (✓ 우리 APK는 stable keystore로 서명됨)
   - 광고 ID 사용 신고가 일치하는지 (✓ 우리는 광고 없음으로 신고)
   - 개인정보처리방침 URL이 살아있는지 (✓ GH Pages에서 운영)
5. **검수 기간**: 보통 3~7 영업일. 거절되면 메일로 사유 옴 → 수정해서 재제출.
